# lobola-co-za-onepager

One-page responsive marketing website for `www.lobola.co.za` (South African marriage/family legal-tech platform). v1 is focused on **market research lead capture** via a modal survey.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react
- Embla Carousel (`embla-carousel-react`)

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Deployment

Vercel-ready defaults (no special config required).
